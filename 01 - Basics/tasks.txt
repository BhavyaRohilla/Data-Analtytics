Rules of Identifiers (Syntax)
-----------------------------

Rule 1: A variable name must start with a letter (a-z, A-Z) or the underscore chracter (_).
Rule 2: A variable accepts only underscore _ symbol only (No other symbol is accepted)
Rule 3: A variable name cannot start with a number
Rule 4: A variable name can contain combination of alphanumeric chracter and underscores 
Rule 5: Variable names are case sensitive (age, Age and AGE are three different variables)
Rule 6: It can't accept reserve/keywords

List of keyword: ['False', 'None', 'True', 'and', 'as', 'assert', 'async', 'await', 'break', 'class', 'continue', 'def', 'del', 'elif', 'else', 'except', 'finally', 'for', 'from', 'global', 'if', 'import', 'in', 'is', 'lambda', 'nonlocal', 'not', 'or', 'pass', 'raise', 'return', 'try', 'while', 'with', 'yield']
